# Mineração de Dados

Os trabalhos são de aulas e tarefas da aula de mineração de dados.

Nas aulas foram ensinados e usadas as bibliotecas:

* csv
* numpy
* pandas
* matplotlib

As aulas consitiram no aprendizado da análise de dados, realizando levantamentos estatísticos sobre esses dados. Dados foram análisados e trabalhados para o entendimentos destes e a criação de gráficos para a visualização dos dados.
